# Algoritmos Greedy - Mochila Fraccional y Cobertura de Antenas

## Descripción
Este proyecto implementa dos problemas clásicos usando algoritmos greedy:

- Mochila fraccional
- Cobertura de antenas

## Tecnologías
- Java

## Estructura del proyecto
/src
  Main.java
  Objeto.java
  MochilaFraccional.java
  CoberturaAntenas.java

## Cómo ejecutar

Compilar:
javac *.java

Ejecutar:
java Main

## Complejidad

- Mochila fraccional: O(n log n)
- Cobertura de antenas: O(n log n)

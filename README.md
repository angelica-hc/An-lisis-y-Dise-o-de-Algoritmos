## Compilación

Para el problema de Escaleras:

```bash
g++ Escaleras.cpp -o Escaleras
./Escaleras
```

Para el problema de Cambio de Monedas:

```bash
g++ cambiomonedas.cpp -o cambiomonedas
./cambiomonedas
```

## Descripción

Este proyecto resuelve dos problemas utilizando Programación Dinámica:

1. Escaleras: calcula el número de formas de subir una escalera de N escalones avanzando 1 o 2 escalones por movimiento.

2. Cambio de Monedas: determina la cantidad mínima de monedas necesarias para formar una cantidad objetivo y muestra una combinación válida junto con la tabla dp.
